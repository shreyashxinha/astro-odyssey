# Astro Odyssey

A modern **2D HTML5 Canvas Space Shooter** built with **Vanilla JavaScript**, featuring responsive desktop and mobile gameplay, endless enemy waves, boss battles, power-ups, particle effects, and a futuristic Neon Cyber interface.

Designed and developed as part of the **Vedam School of Technology Agent Bootcamp**.

---

## 🎮 Live Demo

**Play Here:** https://shreyashxinha.github.io/astro-odyssey/

---

# ✨ Features

### 🚀 Gameplay

* Endless vertical arcade space shooter
* Increasing difficulty over time
* Multiple enemy spaceship types
* Mini Boss & Boss encounters
* Collision detection
* Particle explosions
* Smooth laser shooting
* Lives system
* Score tracking
* Best score saved using LocalStorage
* Game Over & Restart system

---

## ⚡ Power-Ups

Collect randomly dropped power-ups to gain temporary advantages.

Available buffs include:

* ❤️ Extra Life
* ⚡ Speed Boost
* 🔫 Rapid Fire
* 🛡 Energy Shield
* 💥 Damage Boost

---

# 👾 Boss Battles

Bosses appear as the player survives longer and defeats more enemies.

Bosses feature:

* Unique spaceship designs
* Larger health pools
* Different attack patterns
* Health bars
* Boss warning animations

# 📱 Mobile Support

Astro Odyssey is fully playable on mobile browsers.

### Controls

### Desktop

| Action  | Controls                 |
| ------- | ------------------------ |
| Move    | WASD / Arrow Keys        |
| Fire    | Space / Left Mouse Click |
| Pause   | P                        |
| Restart | R                        |

---

### Mobile

* Drag anywhere to move the spaceship
* Auto Fire toggle
* Glassmorphism Fire Button (when Auto Fire is OFF)
* Touch-friendly controls
* Pause / Resume button
* Tap to restart after Game Over
